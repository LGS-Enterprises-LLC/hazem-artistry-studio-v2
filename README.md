# Hazem Artistry Studio

Elite Web Design Agency crafting high-converting websites and sales funnels.

## Tech Stack
This project is built with:
- [Vite](https://vitejs.dev)
- [TypeScript](https://www.typescriptlang.org)
- [React](https://react.dev)
- [shadcn/ui](https://ui.shadcn.com)
- [Tailwind CSS](https://tailwindcss.com)

## Local Development

To run this project locally:

1.  Clone the repository:
    ```sh
    git clone <YOUR_GIT_URL>
    ```

2.  Install dependencies:
    ```sh
    npm install
    ```

3.  Start the development server:
    ```sh
    npm run dev
    ```

## Scripts

-   `npm run dev`: Start development server
-   `npm run build`: Build for production
-   `npm run preview`: Preview production build
-   `npm run lint`: Run ESLint

## Deployment

This project is deployed on [Vercel](https://vercel.com). Pushes to the `main` branch trigger automatic deployments.
